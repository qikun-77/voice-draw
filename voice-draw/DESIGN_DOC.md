# 设计文档：Voice Draw — 纯语音控制绘图工具 v3.0

## 1. 项目概述

Voice Draw 是一款**仅通过语音指令完成绘图创作**的 Web 应用。v3.0 新增：深色网格画布、真实音频波形、弹性动画、PNG导出、快捷键面板、图层列表。

### 技术架构

```
浏览器 (Chrome/Edge)
├── Web Speech API (continuous=true, 手动控制停止)
├── Web Audio API (AnalyserNode → 实时波形)
├── SVG DOM (含 polygon 多边形容器)
├── 状态管理 (对象池 + 操作历史栈 + 图层列表)
│
后端 (Python FastAPI)
├── /api/parse → 混合模式
│   ├── 本地规则引擎 (85%+ 指令, 零API消耗)
│   └── DeepSeek API (复杂物体分解/指代消解)
└── System Prompt 强化 (15色 + 9种角色/物体分解模板)
```

## 3. 已知局限

- Web Speech API 在中文噪音环境下准确率波动
- LLM JSON 输出偶有Markdown包裹, 已通过代码清理
- 复杂角色分解依赖LLM创造力, 质量不完全可控
- `file://` 协议无法使用语音(需 localhost)
